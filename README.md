# ArcLearn

Learn programming languages, frameworks, and tools step-by-step with interactive coding sandboxes and instant checkpoints.

## Features

- 17 programming language guides
- 9 framework guides
- 5 tool guides
- Interactive code sandboxes
- Progress tracking
- Search across all topics
- Tabs for Languages, Frameworks, and Tools

## Tech Stack

- React + Vite
- Zustand for state management
- React Router for navigation

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
guide/
  languages/    - Programming language guides
  frameworks/   - Framework guides
  tools/        - Tool guides
src/
  pages/        - Home, Learn, Search pages
  lib/          - Guide loader
  store/        - Zustand progress store
```
