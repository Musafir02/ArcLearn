Variables

Variables store data that your program can use later. In JavaScript you declare them with let, const, or the old var.

Use const by default. Use let only when the value will change. Avoid var in new code — it has confusing scoping rules.

const name = "Ibrahim";
let score = 0;
score = score + 10;
